# Gemini Skills

This directory is intended for defining custom skills and tools that can be activated and used by the Gemini agent. By organizing reusable scripts and prompts here, you can extend the agent's capabilities for this specific project.

## How it Works

1.  **Define a Skill:** Create a script or a document (e.g., a Python script, a shell script, or a markdown file with instructions) that performs a specific task.
2.  **Activate a Skill:** In your chat with Gemini, you can reference a skill from this directory. The agent can then read the file to understand and execute the defined task.

This approach helps maintain a clean project structure and allows for powerful, project-specific automations.
